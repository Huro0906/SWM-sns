# Mini SNS 프로젝트 범위 및 설계 원칙

| 구분 | 기능 분류 | 상세 기능 | 비고 |
| :--- | :--- | :--- | :--- |
| **필수** | **회원** | 회원가입, 로그인(JWT), 내 정보 조회 | - |
| | **게시글** | 게시글 CRUD (작성/목록/상세/수정/삭제), 내 게시글 조회 | - |
| | **소통** | 댓글 작성/삭제, 좋아요 추가/취소 | - |
| | **조회** | 페이지네이션 | - |
| | **소셜** | 팔로우/언팔로우, 팔로잉 피드 | 선택사항이나 핵심 기능이라 생각되어 추가 |
| **선택** | **알림** | 실시간 알림, 인기글 선정 | MVP구현 후 추가 |
| | **인기** | 인기글 선정 | MVP구현 후 추가 |
| | **검색** | 게시글 키워드 검색 | MVP구현 후 추가 |
| | **이미지** | 이미지 업로드 | MVP구현 후 추가 |
| **제외** | **인프라 확장** | Redis, MQ, Elasticsearch, MSA | 리소스 부족 |
| | **고급 기능** | 추천 알고리즘, OAuth 다중 로그인 | - |
| | **실시간** | 웹소켓(Websocket) 통신 | - |
